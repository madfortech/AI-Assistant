# AI Assistant

An AI-powered assistant for the Godot Editor that helps developers write code, debug scripts, explain engine features, and improve their workflow without leaving the editor.

> **Disclaimer:** This is an independent community plugin and is not affiliated with or endorsed by the Godot Foundation.

---

## ✨ Features

- 🤖 AI-powered assistant directly inside the Godot Editor
- 💬 Interactive chat interface
- 📝 Generate and explain GDScript code
- 🐞 Debug errors and fix common issues
- 🎯 Answer Godot engine and API questions
- 🔑 Secure API Key authentication
- 🌐 Configurable API URL
- 📊 Displays your subscription plan and remaining credits
- ⚡ Fast responses powered by the AI Assistant platform

---

## 🚀 Getting Started

To use the AI Assistant plugin, you'll need an AI Assistant account and a valid API Key.

### 🌐 AI Assistant Platform

**Coming Soon**

The AI Assistant platform will provide:

- User registration
- API Key generation
- Subscription management
- Usage and credit tracking
- Account dashboard

Once available:

1. Create your AI Assistant account.
2. Generate your **Unique API Key**.
3. Copy your **API URL** and **API Key**.
4. Open the **AI Assistant** panel in the Godot Editor.
5. Open **Settings**.
6. Paste your API URL and API Key.
7. Save your settings.
8. Start chatting with the AI Assistant.
